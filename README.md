# Projeto de Sistema Bancário Simples

- Matéria: Fundamentos de Algoritmos
- Semestre: 1°

## Descrição do Projeto

Este projeto consiste em um sistema bancário simples desenvolvido em Python. O sistema oferece funcionalidades básicas de gerenciamento de clientes e operações bancárias, como criação de novos clientes, exclusão de clientes, listagem de clientes, débito, depósito, extrato, transferência entre contas e investimento.

## Funcionalidades

- **Novo Cliente**: Permite adicionar um novo cliente ao sistema.
- **Apagar Cliente**: Permite excluir um cliente existente do sistema.
- **Listar Clientes**: Exibe uma lista de todos os clientes cadastrados no sistema.
- **Débito**: Realiza uma operação de débito na conta de um cliente.
- **Depósito**: Realiza uma operação de depósito na conta de um cliente.
- **Extrato**: Exibe o extrato bancário de um cliente, mostrando todas as operações realizadas.
- **Transferência entre Contas**: Permite transferir dinheiro entre contas de clientes.
- **Investimento**: Permite realizar um investimento financeiro em nome de um cliente.
- **Sair**: Encerra o programa.

## Implementação

O sistema utiliza uma interface de menu para interação com o usuário. Cada opção do menu corresponde a uma função específica do sistema. As operações bancárias são realizadas com base nos dados armazenados em arquivos de texto.

## Como Utilizar

Para utilizar este projeto, siga estas instruções:

1. Execute o programa em um ambiente Python.
2. Use o menu apresentado para selecionar a operação desejada.
3. Siga as instruções apresentadas para fornecer os dados necessários para cada operação.
4. Explore as diferentes funcionalidades do sistema de acordo com suas necessidades bancárias.
